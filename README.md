## Como executar o projeto:
- Clone este repositório com `git clone https://github.com/livialop/biblioteca-bd.git`
- Abra a conexão com o servidor MySQL, acesse o Workbench e execute o comando `create database db_atividade17;`
- Configure a porta no arquivo config/__init__.py ([ou clique aqui](config/__init__.py)).
- ps.: Passo acima é se a sua conexão for diferente da padrão.
- Execute o projeto com `python app.py` (Windows 11), `py app.py` (Windows 10), `python3 app.py` (Linux)
